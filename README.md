# Projeto Final Front — FlameBurger

Este repositório é a entrega final do projeto de front-end — um site de hamburgueria com foco em **acessibilidade (WCAG 2.1 AA)**, **performance** e **boas práticas de versionamento**.

## Estrutura do projeto
```
Projeto_Final_Front/
├── index.html
├── style.css
├── script.js
├── assets/
│   ├── burger1.svg
│   ├── burger2.svg
│   └── burger3.svg
├── dist/
│   ├── index.min.html
│   ├── style.min.css
│   └── script.min.js
├── README.md
└── .gitignore
```

## Como usar
1. Descompacte o ZIP.  
2. Abra `index.html` no navegador ou publique no GitHub Pages (configurar branch `main` com conteúdo).

## Acessibilidade
- Skip link visível ao focar.
- Navegação por teclado em todos os controles.
- Roles e labels ARIA para leitores de tela.
- Modo escuro e modo alto contraste.
- Contraste preservado, foco visível.

## Deploy
Recomenda-se publicar usando **GitHub Pages**:
1. Crie um repositório no GitHub chamado `ProjetoFinalFront` (ou o que preferir).
2. Faça push do conteúdo do ZIP.
3. Ative GitHub Pages na branch `main`.
